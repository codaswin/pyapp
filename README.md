# 🎓 Student Grade Calculator

A modern Streamlit web application that converts student marks into grades.

## Features

- Interactive marks slider
- Beautiful dark theme UI
- Animated gradient button
- Instant grade calculation
- Balloons celebration for Grade A 🎈
- Responsive and user-friendly design

## Grade Scale

| Marks | Grade |
|---------|---------|
| 90 - 100 | A |
| 80 - 89 | B |
| 70 - 79 | C |
| 60 - 69 | D |
| Below 60 | E |

## Technologies Used

- Python
- Streamlit
- CSS Styling

## Run Locally

```bash
pip install streamlit
streamlit run grade_app.py
```
